# SN-Corporation-Dashboard
An interactive and visually rich Power BI dashboard built to analyze and monitor the sales performance of SN Corp from 2017 to 2020. This dashboard provides actionable insights into sales, cost, profit, customer behavior, regional demand, and product performance, helping stakeholders make data-driven decisions.
